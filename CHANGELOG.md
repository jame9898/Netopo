# 更新日志 (Changelog)

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [V1.0.1]  - 2025-03-02

### 新增功能 (Added)
- **PNG 导出功能**：新增导出网络拓扑图为 PNG 图片格式
  - 支持白色背景渲染
  - 自动计算图片尺寸
  - 抗锯齿渲染，保证图片质量
  - 菜单位置：文件 → 导出为PNG...

### 问题修复 (Fixed)
- **PDF 导出功能修复**：解决了 PDF 文件无法正常保存的问题
  - 修复了 PDF 文件导出为 0KB 的问题
  - 修复了网络拓扑图显示为黑色团块的问题
  - 修复了设备清单文字重叠的问题
  - 采用 QImage 中间层方案，确保 PDF 渲染质量
  - 优化了 PDF 页面布局和格式

### 优化改进 (Changed)
- **PDF 导出格式优化**：
  - 网络拓扑图页面：标题位置调整，确保完整显示
  - 设备清单表格：采用 Excel 风格专业表格布局
  - 表格行高和字体大小优化，确保内容清晰可读
  - 文字居中显示，表格在页面中居中
  - 交替行颜色，提高可读性
  - 双向线路记录：每条物理线路生成两条记录（输入/输出）

### 技术改进 (Technical)
- 重构 PDF 导出方法，使用 QImage 中间层
- 优化 QPrinter 配置和渲染流程
- 改进设备清单数据收集逻辑
- 增强错误处理和用户提示

## [V1.0.0] - 2025-03-02

### 新增功能 (Added)
- **设备支持**：
  - 网络设备：交换机、路由器、服务器、防火墙
  - 终端设备：PC、手机、NAS
  - 接入设备：光猫、家用路由器、无线控制器(AC)、无线接入点(AP)
  - 运营商网络：ISP（单接口/双接口）

- **拓扑编辑**：
  - 拖拽式设备添加和移动
  - 设备端口自定义配置（电口、光口、Console、USB）
  - 多种端口布局模式（单行、双行、Z字形、N字形）
  - 设备间端口连接管理
  - 线缆标签显示和管理

- **机架管理**：
  - 47U 标准机架支持
  - 设备上架和 U 位管理
  - 多 U 高度设备支持（1U-8U）
  - 机架视图与拓扑视图联动

- **文件操作**：
  - 保存/加载拓扑文件（.topo 格式）
  - 导出为 PDF 文档（含拓扑图、机架布局、设备清单）

- **用户界面**：
  - 现代化图形界面
  - 工具栏快捷操作
  - 右键菜单交互
  - 设备配置对话框

## 版本说明

- **主版本号（Major）**：不兼容的 API 修改
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正

## 发布周期

- 主版本：重大功能更新或架构变更时发布
- 次版本：新增功能或重要改进时发布
- 修订版：Bug 修复和小幅优化时发布
