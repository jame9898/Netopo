## 主要更新

### 🎉 新增功能

#### PNG 导出功能
新增将网络拓扑图导出为 PNG 图片格式的功能，方便用户分享和文档编写。

**功能特点**：
- 白色背景渲染，适合文档使用
- 自动计算图片尺寸，确保完整显示
- 抗锯齿渲染，保证图片质量
- 支持透明背景（ARGB32格式）

**使用方法**：
1. 创建网络拓扑图
2. 点击菜单：文件 → 导出为PNG...
3. 选择保存路径，输入文件名
4. 点击保存

### 🐛 问题修复

#### PDF 导出功能修复
全面修复了 PDF 导出功能，解决了多个关键问题：

**修复内容**：
1. **PDF 文件为 0KB 问题**
   - 原因：渲染方法与 QPrinter 设备不兼容
   - 解决：采用 QImage 中间层方案

2. **网络拓扑图显示为黑色团块**
   - 原因：场景未设置背景色
   - 解决：为场景设置白色背景

3. **设备清单文字重叠**
   - 原因：表格行高不足，字体过大
   - 解决：优化行高和字体大小

4. **表格格式不规范**
   - 原因：列宽固定，不适应页面
   - 解决：动态计算列宽，居中显示

**技术方案**：
- 使用 QImage 作为渲染中间层
- 场景先渲染到 QImage，再绘制到 PDF
- 避免 QPrinter 坐标系统问题

### ⚡ 优化改进

#### PDF 导出格式优化

**网络拓扑图页面**：
- 标题位置调整，确保完整显示
- 拓扑图与标题间距优化
- 自动缩放适应页面大小

**设备清单表格**：
- Excel 风格专业表格布局

- 交替行颜色（白色/浅灰）
- 清晰的网格线
- 文字居中显示
- 表格在页面中居中

**双向线路记录**：
- 每条物理线路生成两条记录
- 输入线路和输出线路分别显示
- 完整展示设备连接关系

## 安装说明

### 方式一：直接运行 EXE（推荐）

1. 从 [GitHub Releases](https://github.com/your-username/netopo/releases) 或 [Gitee Releases](https://gitee.com/your-username/netopo/releases) 下载 `Netopo.exe`
2. 双击运行即可

### 方式二：从源码运行

安装依赖：

```bash
pip install PySide6
```

运行程序：

```bash
python main.py
```
